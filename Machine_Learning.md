### 이진 크로스 엔트로피(BCE)

최대우도추정(MLE)
그럼 친구의 동전이 진짜 어떤 동전인지 맞춰보려면?
"관측된 결과를 가장 그럴듯하게 만드는 가설을 골라라."
동전 10번 중 앞면 8번 → 가장 그럴듯한 답은 "앞면 확률 = 80%". 왜? 다른 어떤 숫자를 넣어봐도 이 결과에 대한 우도가 이보다 높을 수 없음. 

우도: "이 모델이 내가 본 데이터를 얼마나 그럴듯하게 설명하나?" → 정답들에 준 확률을 전부 곱함
문제: 곱셈은 수치적으로도 수학적으로도 불편
해결: 로그를 씌워서 덧셈으로 바꿈 (로그 우도)
관례: 최대화 대신 최소화가 편하니 마이너스를 붙임 (음의 로그 우도 = BCE)
직관: BCE = 모델이 정답 앞에서 느낀 "놀람의 총합"
학습: BCE를 최소화하는 
w 를 찾는다 = 모델이 덜 놀라게 만든다 = 우도가 최대가 되는 가장 그럴듯한 모델을 찾는다
